# Serverless SQL Studio v1.0.0

Serverless SQL Studio is a privacy-first browser analytics workspace that lets users query local CSV and Parquet files without a backend.

## Highlights

- Runs entirely in the browser with DuckDB-Wasm
- Keeps user files on the local device
- Provides a Monaco-powered SQL editor
- Uses AG Grid for interactive results
- Generates ECharts visualizations
- Supports saved queries, query history, and demo datasets
- Deploys as a static GitHub Pages application

## Recommended repository description
