# Changelog

## v1.0.2 — 2026-07-21

### Fixed

- Replaced the misleading expand icon that only auto-sized AG Grid columns.
- Added a real fullscreen toggle for the complete Results section.
- Added full-screen chart viewing for all visualization tabs.
- Forced AG Grid and ECharts to resize after expanding or restoring.
- Added Escape-key support for leaving fullscreen mode.
- Added separate controls for column auto-sizing and panel expansion.


## v1.0.1 — 2026-07-21

### Fixed

- Added a real SQL formatting provider for Monaco Editor.
- Fixed the nonfunctional Format toolbar button.
- Added visible success and failure feedback after formatting.
- Enabled Monaco's native Format Document action for SQL.


## v1.0.0 — 2026-07-21

First public release.

### Added

- Browser-native DuckDB-Wasm SQL engine
- CSV and Parquet imports
- Monaco SQL editor
- Multiple query tabs
- AG Grid result exploration
- Bar, line, pie, and scatter charts
- PNG chart export
- Result-set statistics
- Saved queries and query history
- Built-in Sales, Manufacturing, and Supply Chain demos
- Guided onboarding tour
- Help Center and keyboard shortcuts
- Browser diagnostics
- GitHub Pages deployment workflow
- Responsive professional UI
